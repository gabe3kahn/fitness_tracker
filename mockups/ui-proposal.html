Arete UI Proposal

Interactive mockup — click the tab bar to switch screens.

Navigation

  • Home — daily driver: hero status, today's stats, sleep bonus, quests
  • Hero — character sheet: hero selector, level/XP, stats, skills
  • Workouts — activity history: calendar heatmap + workout log
  • Profile — all-time stats, health connection, settings

Stats System

  • STR Strength → heavy workouts, HIIT
  • END Endurance → running, cycling, steps
  • DEX Dexterity → yoga, varied, agility
  • INT Intelligence → puzzles, reading
  • LCK Luck → streak consistency

Stats are account-level (not per-hero). Hero class provides a growth bonus to related stats (e.g. Atalanta +20% Endurance gains).

Key Changes vs Current

  • Quests moved into Home (no separate tab)
  • Workouts tab replaces the old duplicate hero/quest overlap
  • Calendar heatmap is now a first-class feature
  • Stats panel is new — surfaces the RPG depth
  • Streak multiplier visible on hero screen
